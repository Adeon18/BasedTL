# BasedTL
My own BASED version of C++ STL (and not only STL) + some custom types. Made in order to learn C++ and save my progress somewhere :)

What is implemented at the moment:
- Containers:
  - [x] vector_d
  - [ ] deque_d
  - [ ] list_d
  - [x] array_d
  - [ ] unordered_map_d

- Iterators
  - [ ] Input
  - [ ] Output
  - [ ] Forward
  - [ ] Bidirectional
  - [x] Random Access

- Algorithms
  - [ ] sort_d
  - [ ] mapreduce_d
  - [ ] copy_d

- Smart Pointers
  - [ ] unique_ptr_d
  - [ ] shared_ptr_d
  - [ ] weak_ptr_d

- Custom:
  - [x] ratio_d - rational number type.
